# OpenClaw Setup Wizard Cheat Sheet

Ziel:
OpenClaw sauber onboarden und stabil betreiben.

Empfohlene Entscheidungen:
1. Personal-by-default bestätigen.
2. QuickStart wählen.
3. Model Provider nach Verfügbarkeit wählen.
4. Telegram als einfachsten Kanal nutzen.
5. Discord später als Alert-/Report-Kanal ergänzen.
6. Web Search nur mit bewusst gewähltem Provider aktivieren.
7. session-memory aktivieren.
8. command-logger aktivieren.
9. Skills minimal halten.
10. Per-skill API Keys nur bei echtem Bedarf setzen.
11. Shell completion akzeptieren.
12. Daemon installieren.

Shadowmaker-Standard:
OpenClaw läuft nicht als lose Terminal-Session, sondern dauerhaft kontrolliert über Stack, Bots, Freeze-Reports und Timeshift.
