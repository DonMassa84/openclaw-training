# Decisions

## 2026-05-02 - Discord als Report-Layer

Telegram bleibt operative Steuerung.
Discord wird strukturierter Report- und Alert-Layer.

## 2026-05-02 - Freeze vor größerer Änderung

Vor jeder größeren Änderung wird /freeze_report ausgeführt.

## 2026-05-02 - Keine freie Shell über Telegram

CrashBandicot erhält nur definierte sichere Befehle.
Kein sudo.
Keine freie Shell.
Keine automatische Löschung.

## 2026-05-02 – Entscheidung: Mnemosyne aktivieren

Entscheidung:
Mnemosyne wird als zweiter echter Agent produktiv gemacht.

Begründung:
Memory-Konsolidierung ist risikoarm, erzeugt direkte Wiederverwendbarkeit und stabilisiert die Multi-Agent-Crew ohne destruktive Rechte.

Grenzen:
Keine Secrets, kein sudo, keine Löschung, keine externen Klartextdaten.

Kontrolle:
mnemosyne-run, latest_mnemosyne_report.md, Discord-Agent-Log.
