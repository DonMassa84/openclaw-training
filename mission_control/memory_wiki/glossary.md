# Glossary

CrashBandicot:
Telegram-Control-Bot für definierte operative Befehle.

Winky:
System-Monitor-Rolle.

Linky:
Builder- und Prompt-to-Code-Rolle.

Mnemosyne:
Memory-Wiki-Rolle.

Daily Delight:
Täglicher kleiner Build- oder Artefaktvorschlag.

Freeze Report:
Sicherer Systemzustandsbericht mit Archiv und SHA256.

Mission Control:
Zentrale Struktur für Aufgaben, Agenten, Projekte, Logs, Memory und Status.
