# Multi Agent Framework Prompt

## Zweck

Entwirft und baut eine Multi-Agent-Crew auf Basis vorhandener OpenClaw-Memory- und Workspace-Daten.

## Schritte

1. Workspace auditieren.
2. Nur fehlende Informationen fragen.
3. Crew von 3 bis 7 Spezialisten vorschlagen.
4. Pro Agent: Name, Rolle, Modell-Tier, Modell, lokal/cloud, Kadenz, Kosten, Privacy.
5. Auf Approval warten.
6. Einen Agenten end-to-end bauen.
7. Funktion beweisen.
8. Danach nächsten Agenten bauen.

## Lokale Shadowmaker-Crew

Aktuell produktiv/manuell:

- Winky: System Monitor
- Mnemosyne: Memory Curator
- Mentor: IHK/MFP Drill
- Strategist: Career/CV/Recruiter
- Steward: Finance/Housing/Jobcenter read-only
- Linky: Builder/Prompt-to-Code
- CrashBandicot: Control/Telegram/Freeze

## Regel

Keine neue Autonomie ohne Nutzenbegründung.
