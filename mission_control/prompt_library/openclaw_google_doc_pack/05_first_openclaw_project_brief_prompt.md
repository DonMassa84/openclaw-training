# First OpenClaw Project Brief Prompt

## Zweck

Hilft, ein erstes sinnvolles OpenClaw-Projekt auszuwählen und als Brief zu formulieren.

## Fragen

1. Nervigste wiederkehrende Aufgabe?
2. Gewünschtes Morgen-Erlebnis?
3. Welche Daten sind verfügbar?
4. Ambition: klein, mittel, groß?

## Starter-Projekte

- Investor Brief
- WhatsApp Triage
- Calendar Defender
- Appointment Scheduler
- Founder Dashboard
- Inbox Executor
- Budget Pulse
- Comment Radar
- Learning Loop

## Ergebnisformat

- Ein Projekt empfehlen
- Projektbrief schreiben
- Zweck, Nutzen, Oberfläche, Daten, Größe, Constraints, Nicht-Spezifikation, Prozess
