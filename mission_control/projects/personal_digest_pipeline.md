
# Personal Digest Pipeline – Shadowmaker Variante

## Zweck

Tägliche Informationsverdichtung für Karriere, AI, OpenClaw, Finanzen, IHK und Systembetrieb.

## Quellen

```text
lokale Logs
Docker Status
Telegram-Bot-Status
OpenClaw Reports
n8n Workflows
Flowise Projekte
manuelle Notizen
```

## Output

```text
1. Systemzustand
2. offene Risiken
3. neue Chancen
4. nächste Aktion
5. was archiviert werden muss
```
