# Phase 3 Operating Rules

## Ziel

Die offenen Video-Bausteine werden modular umgesetzt, ohne das stabile Fundament zu verändern.

## Nicht anfassen

- OpenClaw Binding
- Docker-Port 18789
- Telegram-Core
- Discord-Webhooks-Grundstruktur
- Timeshift-Konfiguration
- Browserprofile
- Secrets / .env / Tokens

## Erlaubt

- Markdown-Module
- read-only Analyse
- Report-Skripte
- Discord-Routing
- Git-Commits
- sichere Timer-Prüfung
- kontrollierte Vorschläge

## Verbot

- Kein sudo in Modulen
- Keine automatische Löschung
- Keine Finanztransaktion
- Kein E-Mail-Schreibzugriff
- Keine Secrets ausgeben
- Keine freie Shell über Telegram
